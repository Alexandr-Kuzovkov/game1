Это файл справки по использованию модуля высотных данных основанного на сервисе высотных данных Google Maps

  • http://localhost:8000/maps/api/elevation/json?lat=75&lng=134 - пример запроса для получения высоты точки с заданными координатами
  • http://localhost:8000/maps/api/elevation/multi/json - пример запроса для получения высот массива точек.
    Метод запроса: POST,
    Тип данных: application/x-www-form-urlencoded
    Пример тела запроса: obj={'path':[[34,45],[34.5,46],[35,47],[56,88.005],[56,89],[61,28]]}
  • localhost:8000/help - вызов данной справки
  • {"status":"Ok","lat":"75","lng":"134","elevation":-19.5305290222168} - пример успешного результата запроса
  • {"status":"Ok",[{"lat":"74","lng":"133","elevation":-18.5303230242168},{"lat":"75","lng":"134","elevation":-19.5305290222168},{"lat":"76","lng":"136","elevation":-17.53651726168}]} - пример успешного результата запроса по массиву точек
  • {"status":"Fail"} - пример неуспешного результата запроса
